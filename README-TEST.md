# CouchMate Core 1.2.0-alpha.10

- API liefert Auswahlmodell-v2-Metadaten an tvOS.
- `explicit_entity_ids` ist die exakte Whitelist einzelner Funktionen.
- `full_device_ids` enthält ausschließlich Geräte mit „Ganzes Gerät verwenden“.
- tvOS kann dadurch Geschwister- und Diagnoseentitäten zuverlässig ausschließen.
